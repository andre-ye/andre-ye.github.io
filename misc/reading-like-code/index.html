The parser prompt

Paste this into Claude (or any strong model), then drop in any dense piece of writing. It returns one self-contained HTML page that renders the exact text — never reworded — as a typed, colored, bracketed structure with a clickable concept index.

1. Copy the prompt below.   2. Replace [[[ PASTE YOUR TEXT HERE ]]] at the very end with your text.   3. Send it.   4. Save the model's reply as .html and open it.

Built for working through philosophy, theory, and other tightly-argued prose. See a finished example: Brandom, A Spirit of Trust, ch. 11 ↗
